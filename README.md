# ぼくの・わたしのものがたり 📖

お子様向けの物語生成アプリです。
好きな人物や友達、物語のジャンルを入力することで
世界に一つだけのオリジナルの物語を作ることができます。

---

## 設計思想

お子様、親御さんにとって使いやすい設計を心がけました。

・ひらがな中心のUI
・絵本風のページめくり
・やわらかい色使い（グリーン系パステル）
・ジャンル選択でこどもが好きなテーマを選べる
・画像をページごとに表示
おやすみ前のワンシーンにぜひご利用ください。

## 機能一覧

- **ものがたりさくせい** — ジャンル・しゅじんこう・なかま・てき・アイテムを入力して、オリジナルのものがたりを生成
- **ページめくり表示** — 絵本風にページをめくりながら読める
- **みんなのものがたり一覧** — ほかのひとのつくったものがたりをみることができる
- **いいね機能** — きにいったものがたりにいいねができる
- **15ジャンル対応** — ぼうけん・まほう・せんたい・にんじゃ・ロボットなど

## ものがたり生成の仕組み

```
① ユーザーがフォームに入力（しゅじんこう・なかま・てき・アイテム・ジャンル）
      ↓
② 選択されたジャンルのテンプレートをランダム取得
      ↓
③ str_replace でプレースホルダーを入力値に置き換え
   例: {主人公} → はちわれ
       {なかま} → ちいかわ
       {てき}   → でかつよ
      ↓
④ 絵本風ページめくり画面で表示
```

## 画像について

各ジャンルの画像は `public/images/genres/{ジャンル名}/` に配置します。

```
cover.png      （表紙）
intro.png      （はじまり）
develop.png    （つぎに）
conversion.png （そして）
ending.png     （おわり）
back.png       （裏表紙）
```
